Periscope Lights Installer

Installer for Periscope Lights firmware on ESP32 devices.

  • You need an FTDI programmer.
  • Connect 5V and GND as usual.
  • Cross connect RX and TX (RX goes to TX and vice versa)
  • Press and hold Flash button
  • Press Reset button
  • Release Flash button
  • Select desired firwmare below. Select either Uppity Spinner (ABC pins) or Serial control.
  • Click Connect to Device
  • Select serial port
  • Click Connect
  • Select INSTALL PRINTEDDROID

If you installed serial firmware you can select LOGS & CONSOLE. These are the valid serial commands:

  • :PL0 - Random
  • :PL1 - Off
  • :PL2 - Obi-Wan
  • :PL3 - Yoda
  • :PL4 - Sith
  • :PL5 - Search-Light
  • :PL6 - Dagobah
  • :PL7 - Strobe
  • ON - Random
  • OFF - Off
